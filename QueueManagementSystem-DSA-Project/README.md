# Queue-Based Ticket Reservation System

A Java application for managing ticket reservations using queue-based data structures and MySQL database.

## Overview
This project implements a queue-based ticket reservation system where customers can reserve tickets in a FIFO (First-In-First-Out) manner.

## Technologies Used
- Java
- MySQL Database
- Queue Data Structure

## Project Structure
- `src/`: Source code directory
- `database/`: Database schema and sample data
- `diagrams/`: System diagrams (UML, sequence diagrams)
- `docs/`: Project documentation
- `lib/`: External libraries (MySQL Connector)

## How to Run
1. Set up the MySQL database using `database/schema.sql`
2. Compile the Java files
3. Run the `Main.java` class
